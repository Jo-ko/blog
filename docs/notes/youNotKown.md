# 你不知道的JavaScript
## 作用域
**一句话描述作用域: LHS和RHS的读取环境**
### 词法作用域
也叫静态作用域,就是词法解析时的作用域, 换句话说, 词法作用域是由你在写代码时 __将变量和块作用域写在哪里来决定的__
### 作用域欺骗
**注意: 欺骗词法作用域会导致性能下降和变量污染!**
1. eval
2. setTimeout/setInterval
3. new Function()
4. with

::: tip 性能下降
大多数的JS引擎都会根据代码的词法做一些静态分析,优化执行的代码,欺骗词法作用域类似于动态作用域,这样就很难静态分析
:::
### 词法作用域两种类型
1. 函数作用域
2. 块级作用域
